# 11 — Flask Web Basics

A minimal Flask app: routes, templates, and handling requests.

## Files
- `app.py` — the Flask application
- `templates/index.html`, `templates/greet.html` — Jinja2 templates
- `requirements.txt`

## Run
```bash
pip install -r requirements.txt
python3 app.py
# visit http://127.0.0.1:5000
```
