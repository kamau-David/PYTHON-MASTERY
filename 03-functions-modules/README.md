# 03 — Functions & Modules

Function definitions, arguments, scope, and organizing code into modules.

## Files
- `functions.py` — default args, *args/**kwargs, return values, scope
- `math_utils.py` — a small module to import from
- `use_module.py` — demonstrates importing your own module
- `main.py` — entry point tying it together

## Run
```bash
python3 functions.py
python3 use_module.py
python3 main.py
```
