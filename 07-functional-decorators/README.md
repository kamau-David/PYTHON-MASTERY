# 07 — Functional Programming & Decorators

Lambdas, map/filter/reduce, decorators, and generators.

## Files
- `lambda_map_filter.py` — lambda expressions, map, filter, reduce, sorted with key
- `decorators.py` — writing and stacking your own decorators
- `generators.py` — generator functions, `yield`, lazy pipelines

## Run
```bash
python3 lambda_map_filter.py
python3 decorators.py
python3 generators.py
```
