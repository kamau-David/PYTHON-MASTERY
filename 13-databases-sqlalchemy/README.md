# 13 — Databases with SQLAlchemy

Defining ORM models, running queries, and relationships - using SQLite so
there's nothing extra to install or configure.

## Files
- `models.py` — SQLAlchemy ORM models with a relationship
- `main.py` — creating tables, inserting, querying, updating, deleting
- `requirements.txt`

## Run
```bash
pip install -r requirements.txt
python3 main.py
```
