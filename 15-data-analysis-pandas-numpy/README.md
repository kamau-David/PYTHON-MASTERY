# 15 — Data Analysis with NumPy & Pandas

Arrays, vectorized operations, DataFrames, cleaning, and aggregation.

## Files
- `numpy_basics.py` — arrays, vectorized math, broadcasting, indexing
- `pandas_basics.py` — DataFrames, filtering, groupby, merging
- `data_cleaning.py` — handling missing data, duplicates, type conversion
- `sample_sales.csv` — sample dataset used by the pandas scripts
- `requirements.txt`

## Run
```bash
pip install -r requirements.txt
python3 numpy_basics.py
python3 pandas_basics.py
python3 data_cleaning.py
```
