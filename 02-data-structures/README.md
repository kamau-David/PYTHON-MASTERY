# 02 — Data Structures

Python's built-in collections and comprehensions.

## Files
- `lists_tuples.py` — lists, tuples, slicing, mutability
- `dicts_sets.py` — dictionaries, sets, membership testing
- `comprehensions.py` — list/dict/set comprehensions, generator expressions

## Run
```bash
python3 lists_tuples.py
python3 dicts_sets.py
python3 comprehensions.py
```
