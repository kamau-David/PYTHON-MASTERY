# 04 — OOP Basics

Classes, instances, inheritance, and dunder (magic) methods.

## Files
- `classes_basics.py` — class definition, `__init__`, instance methods
- `inheritance.py` — subclassing, `super()`, method overriding
- `dunder_methods.py` — `__str__`, `__eq__`, `__len__`, operator overloading

## Run
```bash
python3 classes_basics.py
python3 inheritance.py
python3 dunder_methods.py
```
