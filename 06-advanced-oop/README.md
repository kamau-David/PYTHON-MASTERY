# 06 — Advanced OOP

Dataclasses, abstract base classes, mixins, and properties.

## Files
- `dataclasses_demo.py` — `@dataclass` for boilerplate-free data containers
- `abstract_classes.py` — `ABC`, `@abstractmethod`, enforcing an interface
- `properties_mixins.py` — `@property` for computed attributes, mixin classes

## Run
```bash
python3 dataclasses_demo.py
python3 abstract_classes.py
python3 properties_mixins.py
```
