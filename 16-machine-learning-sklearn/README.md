# 16 — Machine Learning with scikit-learn

Train/test splits, a regression model, a classification model, and proper
evaluation - the standard scikit-learn workflow.

## Files
- `regression_demo.py` — predicting a continuous value (house price)
- `classification_demo.py` — predicting a category (iris species)
- `pipeline_and_evaluation.py` — preprocessing pipelines, cross-validation
- `requirements.txt`

## Run
```bash
pip install -r requirements.txt
python3 regression_demo.py
python3 classification_demo.py
python3 pipeline_and_evaluation.py
```
