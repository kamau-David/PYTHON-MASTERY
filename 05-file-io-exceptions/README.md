# 05 — File I/O & Exceptions

Reading/writing files, exception handling, and context managers.

## Files
- `file_operations.py` — reading/writing text and JSON files
- `exceptions.py` — try/except/else/finally, custom exceptions
- `context_managers.py` — `with` statement, writing your own context manager

## Run
```bash
python3 file_operations.py
python3 exceptions.py
python3 context_managers.py
```
