# 01 — Python Fundamentals

The absolute basics: variables, core types, operators, and control flow.

## Files
- `variables_types.py` — variables, dynamic typing, type conversion
- `operators.py` — arithmetic, comparison, logical, walrus operator
- `control_flow.py` — if/elif/else, for/while loops, break/continue/else

## Run
```bash
python3 variables_types.py
python3 operators.py
python3 control_flow.py
```
